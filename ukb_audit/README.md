# UKB-Git-Audit
Prior to publishing this repository, we tested for potential sensitive information from the UKB using the [UKB-Git-Audit-Tool](https://github.com/UK-Biobank/UKB-Git-Audit-Tool). 
Both csv files are the output of this tool to report that there is no sensitive information shared. The only two IDs reported there are coming from Jupyter-Notebook metadata.
